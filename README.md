## 4. 주요 기능

* 회원가입, 로그인
    * JWT를 활용한 로그인 인증
* 글 작성
    * 게시글 및 댓글 CRUD
    * 게시글에 태그 추가, 이미지 첨부 가능.
    * 제목으로 검색하기
    * 태그로 검색하기
    * 내 벨로그에서 내가 쓴 글 모아보기
    * 좋아요 등록 / 취소
    * 좋아요 누른 글 모아보기
* 설정 페이지 (마이페이지)
    * 프로필 이미지, 자기소개 문구, 벨로그 제목 수정
